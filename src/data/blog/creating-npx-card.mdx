# Prequistes
Before making the card, you will need few things to started with.
- [Node](https://nodejs.org/en/)
- [NPM](https://www.npmjs.com/)

# Starting of
Now as you have everything set up, its time to code!

You can check out my repository [here](https://github.com/BruceMacGary/npx-brucemacgary).

- Clone my repository locally, by typing `git clone https://github.com/BruceMacGary/npx-brucemacgary.git`
- Type `npm i` to install all the packages.
- Make changes as per yourself in the `index.js`.
- At last, test your changes by typing `node .`

## Publishing to NPM
You must have a verified NPM account, i.e. a verified email on your account in order to publish the npm package.

- Type `npm login` to login to your account.
- Type `npm publish` to publish your npm package!
- The name in the package json file will be your npx name.
